# Markdown & Obsidian Syntax

## H 제목 (Headers)

| 설명 (Description) | 단축키 / 문법 (Shortcut) |
|:---|:---|
| 가장 큰 제목 (H1) | `# 제목` |
| 두 번째 큰 제목 (H2) | `## 제목` |
| 세 번째 큰 제목 (H3) | `### 제목` |
| 크기별 제목 (H1~H6) | `#~###### 제목` |

## ✨ 강조 (Emphasis)

| 설명 (Description) | 단축키 / 문법 (Shortcut) |
|:---|:---|
| 기울임꼴 (Italic) | `*텍스트*` |
| 기울임꼴 (언더스코어) | `_텍스트_` |
| 굵게 (Bold) | `**텍스트**` |
| 굵고 기울임꼴 | `***텍스트***` |
| 취소선 (Strikethrough) | `~~텍스트~~` |

## 📋 목록 (Lists)

| 설명 (Description) | 단축키 / 문법 (Shortcut) |
|:---|:---|
| 순서 없는 목록 (Unordered) | `- 항목` |
| 순서 없는 대체 기호 | `* 항목` |
| 순서 있는 목록 (Ordered) | `1. 항목` |
| 할 일 목록 (빈 체크박스) | `- [ ] 할일` |
| 할 일 목록 (완료) | `- [x] 완료` |

## 🔗 링크 & 이미지 (Links & Images)

| 설명 (Description) | 단축키 / 문법 (Shortcut) |
|:---|:---|
| 링크 (Link) | `[이름](URL)` |
| 이미지 삽입 (Image) | `![대체텍스트](URL)` |
| 제목 링크 (참조형) | `[이름][id]` |
| 자동 링크 | `<http://주소>` |

## 💻 코드 & 인용 (Code & Quotes)

| 설명 (Description) | 단축키 / 문법 (Shortcut) |
|:---|:---|
| 인라인 코드 (Inline) | ``코드`` |
| 코드 블럭 (Block) | ````코드```` |
| 언어 지정 코드 블럭 | ````js` |
| 블록 인용구 (Blockquote) | `> 인용문` |
| 중첩 인용구 | `>> 중첩문` |

## 📊 구분선 및 표 (Tables & Lines)

| 설명 (Description) | 단축키 / 문법 (Shortcut) |
|:---|:---|
| 수평선 (가로줄) | `---` |
| 수평선 대체 | `***` |
| 표 (Table) 항목 구분 | `| 항목 | 항목 |` |
| 표 정렬 (오른쪽) | `| ---: |` |
| 표 정렬 (가운데) | `| :---: |` |

## ⚡ 기타 (Misc & Extensions)

| 설명 (Description) | 단축키 / 문법 (Shortcut) |
|:---|:---|
| 이스케이프 기호 (무시) | `\*` |
| 각주 (Footnotes) | `[^1]` |
| 줄바꿈 (Line break) | `스페이스 2번 Enter` |

## 💎 옵시디언 콜아웃 (Obsidian)

| 설명 (Description) | 단축키 / 문법 (Shortcut) |
|:---|:---|
| 기본 메모 (Note) | `> [!note]` |
| 요약 (Abstract/Summary) | `> [!abstract] / [!summary] / [!tldr]` |
| 정보 (Info) | `> [!info]` |
| 할일 (Todo) | `> [!todo]` |
| 팁/중요 (Tip/Important) | `> [!tip] / [!hint] / [!important]` |
| 성공/완료 (Success/Done) | `> [!success] / [!check] / [!done]` |
| 질문/도움말 (Question/FAQ) | `> [!question] / [!help] / [!faq]` |
| 경고/주의 (Warning/Caution) | `> [!warning] / [!caution] / [!attention]` |
| 실패/누락 (Failure/Missing) | `> [!failure] / [!fail] / [!missing]` |
| 위험/에러 (Danger/Error) | `> [!danger] / [!error]` |
| 버그 (Bug) | `> [!bug]` |
| 예시 (Example) | `> [!example]` |
| 인용구 (Quote/Cite) | `> [!quote] / [!cite]` |
