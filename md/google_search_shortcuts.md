# Google Search Commands

## 🔍 기본 연산자 (Basic)

| 설명 (Description) | 단축키 / 문법 (Shortcut) |
|:---|:---|
| 정확한 단어/문구 포함 | `"검색어"` |
| 둘 중 하나 검색 (OR) | `단어 OR 단어 + 단어 | 단어` |
| 모두 포함 (AND) | `단어 AND 단어` |
| 특정 검색어 제외 | `-제외단어` |
| 와일드카드 (아무 단어) | `*` |
| 공통 키워드 그룹화 | `(단어 OR 단어)` |

## 🛠️ 특정 속성 & 도구 (Properties)

| 설명 (Description) | 단축키 / 문법 (Shortcut) |
|:---|:---|
| 가격 검색 | `$329 + €100` |
| 단위/환율 변환 | `$1000 in KRW` |
| 단어 사전 정의 | `define:단어` |
| 캐시된 페이지 확인 | `cache:URL` |
| 특정 확장자/파일타입 | `filetype:pdf + ext:pdf` |

## 🌐 웹사이트 & URL (Site & URL)

| 설명 (Description) | 단축키 / 문법 (Shortcut) |
|:---|:---|
| 특정 웹사이트 내 검색 | `site:도메인` |
| 비슷한 성격의 웹사이트 | `related:도메인` |
| URL에 특정 단어 포함 | `inurl:단어` |
| URL에 모든 단어 포함 | `allinurl:단어들` |

## 📝 본문 & 타이틀 (Text & Title)

| 설명 (Description) | 단축키 / 문법 (Shortcut) |
|:---|:---|
| 타이틀 태그에 포함 | `intitle:단어` |
| 타이틀에 모든 단어 포함 | `allintitle:단어들` |
| 본문(Body)에 포함 | `intext:단어` |
| 본문에 모든 단어 포함 | `allintext:단어들` |
| X 단어수 이내 근접 | `단어 AROUND(3) 단어` |

## 🎬 특정 콘텐츠 검색 (Content)

| 설명 (Description) | 단축키 / 문법 (Shortcut) |
|:---|:---|
| 지역 날씨 검색 | `weather:위치` |
| 주식 정보 (티커) | `stocks:현대차` |
| 지도/지역 검색 | `map:위치` |
| 영화 관련 검색결과 | `movie:영화명` |
| 뉴스 출처 검색 | `source:언론사명` |

## ⚠️ 기타/부분작동 명령어 (Misc)

| 설명 (Description) | 단축키 / 문법 (Shortcut) |
|:---|:---|
| 숫자/날짜 범위 (정확도↓) | `2020..2021` |
| 앵커텍스트에 단어 포함 | `inanchor:단어` |
| 모든 앵커텍스트에 포함 | `allinanchor:단어들` |
| 특정 도메인의 블로그 URL | `blogurl:도메인` |
| 특정 위치 (일반/뉴스) | `loc:위치 + location:위치` |
