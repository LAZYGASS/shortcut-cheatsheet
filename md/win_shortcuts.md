# Windows OS Shortcuts

## ⚙️ 기본 및 시스템 (System)

| 설명 (Description) | 단축키 / 문법 (Shortcut) |
|:---|:---|
| 파일 탐색기 열기 | `Win + E` |
| Windows 설정 열기 | `Win + I` |
| PC 잠금 (화면 잠금) | `Win + L` |
| 대화 상자 열기 (실행) | `Win + R` |
| 고급 시작 메뉴 열기 | `Win + X` |
| 접근성/디스플레이 설정 | `Win + U` |

## 🖥️ 창 관리 (Window Management)

| 설명 (Description) | 단축키 / 문법 (Shortcut) |
|:---|:---|
| 바탕 화면 표시/숨기기 | `Win + D` |
| 모든 창 최소화 | `Win + M` |
| 마우스 커서 가져다대기 (바탕화면 엿보기) | `Win + ,` |
| 화면 분할 / 창 고정/최대화 | `Win + 방향키` |
| 보조 모니터로 창 이동 | `Win + Shift + 방향키` |
| 작업 보기 (Task View) | `Win + Tab` |

## 📌 작업 표시줄 및 앱 (Taskbar & Apps)

| 설명 (Description) | 단축키 / 문법 (Shortcut) |
|:---|:---|
| 고정된 앱 순서대로 실행 | `Win + 1~9` |
| 작업 표시줄 앱 전환 | `Win + T` |

## 🛠️ 유틸리티 및 도구 (Utilities)

| 설명 (Description) | 단축키 / 문법 (Shortcut) |
|:---|:---|
| 클립보드 내역 열기 | `Win + V` |
| 스크린샷 도구 (영역 지정) | `Win + Shift + S` |
| 전체화면 스크린샷 (자동저장) | `Win + PrtScn` |
| 이모지/기호 패널 열기 | `Win + .` |
| 음성 입력 활성화 | `Win + H` |
| 입력 언어/방법 전환 | `Win + Space` |
| 디스플레이 프로젝트/확장 설정 | `Win + P` |
