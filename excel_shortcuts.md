# Excel Shortcuts (Windows)

## ⭐ 핵심 단축키 (Essential)

| 설명 (Description) | 단축키 / 문법 (Shortcut) |
|:---|:---|
| 가운데 정렬 | `Alt + H + A + C` |
| 모든 테두리 그리기 | `Alt + H + B + A` |
| 테두리 색상(회색) 변경 | `Alt + H + B + I` |
| 통합문서 저장 | `Ctrl + S` |
| 작업 취소 (Undo) | `Ctrl + Z` |
| 다시 실행 (Redo) | `Ctrl + Y` |
| 찾기 | `Ctrl + F` |
| 바꾸기 | `Ctrl + H` |
| 셀 콘텐츠 삭제 | `Delete` |
| 바로 가기 메뉴 열기 | `Shift + F10` |
